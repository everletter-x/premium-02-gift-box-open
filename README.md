# Premium 02 - Gift Box Open

EverLetter premium template featuring an animated gift box that opens to reveal personalized content.

## Features

- Animated gift box with spring physics
- Click-to-open interaction
- Light burst effect on reveal
- Staggered content animation
- Theme support (pink, lavender, gold)
- Music floating button
- Fully configurable via `public/config.json`

## Getting Started

```bash
pnpm install
pnpm dev
```

## Configuration

Edit `public/config.json` to customize:

- `recipient` - Name of the recipient
- `sender` - Name of the sender
- `title` - Main title
- `message` - Personal message
- `photos` - Array of photo filenames
- `theme` - Color theme (pink/lavender/gold)
- `music` - Music filename
- `captions` - Photo captions
- `closing` - Closing message

## Build

```bash
pnpm build
pnpm start
```
